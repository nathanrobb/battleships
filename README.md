# battleships

Basic battleships game implemented in WebAPI

## To Develop

### App

1. `cd Battleships`
1. `npm install`

## To run

Requires `node 14.x` and `dotnet core 3.1`

1. Run app
   1. `cd Battleships`
   1. `npm install`
   1. `npm start`
1. Run api
   1. `cd Api.BattleShips`
   1. `dotnet run`
